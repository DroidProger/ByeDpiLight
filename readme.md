 # ByeDpiLight for Android

Еще одно приложение запускающее [byedpi](https://github.com/hufrea/byedpi) на android. Поддерживает только vpn режим и передачу параметров только через командную строку. Использует [hev-socks5-tunnel](https://github.com/heiher/hev-socks5-tunnel) для направления всего трафика на byedpi. Большая часть кода взята от [ByeDPIAndroid](https://github.com/dovecoteescapee/ByeDPIAndroid).

Об аргументах командной строки можно прочитать в readme [byedpi](https://github.com/hufrea/byedpi).
В настройках hev-socks5-tunnel необходимо указывать такой же порт как в настройках byedpi. Dns можно указать любой, обычно это 1.1.1.1 или 8.8.8.8
